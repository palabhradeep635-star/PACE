# 🌌 Overview

PACE is an AI-powered academic and competitive programming platform designed to bring every aspect of a student's learning journey into one place.

Instead of tracking coding, academics, projects, and study habits across multiple disconnected applications, PACE unifies them into a single intelligent dashboard.

The platform combines:

- 📚 Academic Progress
- 💻 Competitive Programming
- 🚀 Projects
- 👥 Social Learning
- 🤖 AI Analytics
- 🏆 Gamification

to create a real-time learning operating system.

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/PACE.git
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```
